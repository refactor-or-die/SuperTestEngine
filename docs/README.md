# Super Text Engine

Projekt realizowany przez Stanisława Kucharskiego i Diega Ostoję-Kowalskiego.

Projekt NIE jest współfinansowany ze środków Unii Europejskiej.

## Cel aplikacji

Aplikacja Super Text Engine ma być prostym edytorem tekstu. Jej dwie najważniejsze przewagi względem prototypowego notatnika to:

* zapisywanie tekstu bogatego jako odpowiadający mu tekst prosty;
* cofanie i ponawianie zmian w pliku.

## Diagram UML

Prototypowy diagram klas znajduje się w `/docs/uml/diagram.svg`.

## Implementacja

Aplikacja będzie napisana za pomocą języka programowania Java i biblioteki JavaFX.
